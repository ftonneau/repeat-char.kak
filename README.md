# repeat-char.kak

This plugin for the Kakoune editor allows you to insert a character _n_ times in Insert mode.
Character repetition is enabled with the `repeat-char-enable-on <key>` command, which you may want
to add to your `kakrc` (replacing `<key>` by the key combination of your choice, for example, `
<a-r>`).

Character repetition can be disabled at any time with the `repeat-char-disable` command.

## Installation

Put the `repeat-char.kak` file in your autoload directory (or in one of its subdirectories).
